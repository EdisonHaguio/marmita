# Sistema Dona Guedes - Marmitaria

## Como Instalar (Windows)

1. Instale: [Python](https://python.org) e [Node.js](https://nodejs.org)
2. Execute: `INSTALAR.bat`
3. Execute: `INICIAR.bat`
4. Acesse: http://localhost:3000

**Login:** admin / admin123

---

## Funcionalidades

- Criar pedidos com múltiplas marmitas
- Marmita P = 1 proteína | M e G = 2 proteínas (2ª grátis)
- Pagamento: Dinheiro (com troco), PIX, Cartão, Fiado
- Imprimir cupom (ESC/POS ou Windows)
- Reimprimir 2ª via
- Relatório de vendas diárias
- Pedidos para empresas (cupom individual por funcionário)
